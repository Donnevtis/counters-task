# Секундомеры

## Установить

```
npm install
```

### Скомпилировать для разработки

```
npm run serve
```

### Скомпилировать для продакшна

```
npm run build
```

### Запустить юнит тесты

```
npm run test:unit
```

### Запустить линтер

```
npm run lint
```
